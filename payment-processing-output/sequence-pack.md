# Sequence Pack — avery-pay-platform

> Email and outreach sequences for onboarding, retention, and re-engagement

## Welcome Sequence (Post-Install)

### Email 1: Welcome (Day 0)

**Subject**: Welcome to avery-pay-platform — here's your quickstart

**Body**:
- Brief welcome and what avery-pay-platform does
- Link to quickstart guide
- One concrete example they can try in 2 minutes
- CTA: Try the quickstart

### Email 2: Core Feature (Day 2)

**Subject**: The one avery-pay-platform feature everyone uses first

**Body**:
- Highlight the core feature: **go-backend/ (project_directory)**
- Step-by-step walkthrough with code/screenshots
- CTA: Try this feature

### Email 3: Power User Tip (Day 5)

**Subject**: Level up your avery-pay-platform usage

**Body**:
- Advanced tip or lesser-known feature
- Pro convention: TypeScript strict mode
- Link to documentation or example repo
- CTA: Explore advanced docs
